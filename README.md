# 零壹对话总结助手 - Chrome 插件

一个用于自动扫描并总结页面中对话信息的 Chrome 浏览器插件，固定在浏览器右侧侧边栏。

**开发人：零壹**

## 功能特点

- 📋 自动扫描页面中具有 `data-virtual-list-item-key` 属性的元素
- 🎯 点击列表项快速跳转到对应位置并高亮显示
- 🔄 支持手动刷新扫描，无需刷新页面
- 🔍 支持搜索过滤问题
- 🎨 美观的侧边栏 UI 设计
- ⚡ 实时监听 DOM 变化自动更新

## 安装步骤

1. 打开 Chrome 浏览器，进入扩展程序管理页面
   - 地址栏输入：`chrome://extensions/`
   - 或者：菜单 → 更多工具 → 扩展程序

2. 开启右上角的 **开发者模式**

3. 点击 **加载已解压的扩展程序**

4. 选择本项目的文件夹 `d:\plugin`

5. 安装完成！插件图标会出现在浏览器工具栏中

## 使用方法

1. 打开包含提问内容的网页
2. 点击浏览器工具栏中的插件图标
3. 在弹出的窗口中查看提问列表
4. 点击任意问题项，页面会自动滚动并高亮显示该问题

## 问题识别规则

插件会自动识别具有以下属性的元素：

- `data-virtual-list-item-key` 属性

## 注意事项

- 首次加载页面时可能需要等待几秒钟进行扫描
- 如果页面内容动态加载，插件会自动重新扫描
- 如果点击后没有反应，请刷新页面后重试

## 开发说明

- Manifest Version: 3 (Chrome 最新标准)
- 使用原生 JavaScript，无外部依赖
- 支持热重载：在 `chrome://extensions/` 点击刷新按钮即可
